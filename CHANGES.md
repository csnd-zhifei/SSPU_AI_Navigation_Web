# 更新日志

## 2026年3月7日 - 新增功能

### 1. 核心交互改进
- **中央导航区图标**：点击图标现在直接跳转到网站，而不是打开分类菜单
- **分类标题可点击**：点击分类标题（如"校园服务"）打开该分类的详细菜单
- **删除功能**：dock中的图标和"我的收藏"中的用户添加图标支持右键删除

### 2. 背景设置增强
#### 纯色背景选择
- 6个预设颜色（白色、浅灰、灰色、深灰、蓝色、青色）
- 自定义颜色选择器，可选择任意颜色
- 选择纯色时会自动清除图片背景

#### 模糊背景效果
- 新增"启用模糊效果"开关
- 打开时，所有卡片显示毛玻璃效果（backdrop-filter: blur）
- 关闭时，卡片背景透明清晰

#### 壁纸画廊
- 新增"我的壁纸"画廊，显示所有已上传的壁纸
- 点击画廊中的壁纸缩略图即可应用为当前背景
- 支持图片悬停效果预览

### 3. 后端改进
- 新增 `/api/wallpaper/list` 接口
- 返回服务器上所有已上传的壁纸列表
- 支持jpg、png、webp图片格式

### 4. 数据持久化
所有设置项已保存到localStorage：
- `background_color`: 当前纯色背景颜色
- `blur_enabled`: 模糊效果是否开启（true/false）
- `wallpaper_url`: 当前壁纸URL
- 其他现有设置同步更新

## 使用方法

### 更换背景
1. 打开设置面板（右上角齿轮图标）
2. 在"壁纸设置"中选择：
   - **纯色背景**：点击预设颜色或使用自定义色选择器
   - **图片背景**：从"我的壁纸"画廊选择，或上传新壁纸，或AI生成

### 调整模糊效果
1. 打开设置面板
2. 在"壁纸设置"中找到"启用模糊效果"开关
3. 打开/关闭毛玻璃效果

### 删除不需要的图标
1. 在dock（底部快捷方式）或"我的收藏"中找到要删除的图标
2. 右键点击该图标
3. 选择"删除"
4. 确认删除

## 技术细节

### 修改的文件
- `app.py`: 添加wall壁纸列表API
- `templates/index.html`: 
  - 新增设置面板UI组件
  - 新增JavaScript函数处理颜色、模糊、壁纸选择
  - 更新DOMContentLoaded初始化逻辑

### 新增函数
- `applyBackgroundColor()`: 应用纯色背景
- `applyBlurEffect()`: 打开/关闭毛玻璃效果
- `loadWallpaperGallery()`: 加载并显示壁纸画廊

### localStorage数据结构
```javascript
{
  show_tags_area: true,          // 显示分类卡片
  enabled_engines: ['baidu'...], // 启用的搜索引擎
  hotkeys_enabled: true,         // 快捷键
  wallpaper_url: '',             // 背景图片URL
  background_color: '',          // 纯色背景色值
  blur_enabled: true             // 模糊效果
}
```
