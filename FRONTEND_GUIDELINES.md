# 前端设计规范

## 1. 配色方案
- **背景色**: 默认浅灰/白 (#F3F4F6)，跟随壁纸。
- **卡片/Dock背景**: 纯白 (#FFFFFF) 或 半透明白 (rgba(255,255,255,0.8))。
- **文字**:
  - 主文字: 黑 (#1F2937)
  - 次文字: 灰 (#6B7280)
- **强调色 (标签/按钮)**:
  - 百度: 蓝色 (#2932E1)
  - 必应: 绿色 (#008373)
  - 谷歌: 红色 (#EA4335)
  - AI问答: 深蓝色 (#4F46E5)
- **高亮色**: 红色背景 (rgba(239, 68, 68, 0.3))
- **遮罩层**: rgba(0, 0, 0, 0.5)

## 2. 布局规则

### 2.1 顶部搜索区
- **位置**: 垂直居中偏上，距离顶部约 15vh。
- **搜索框尺寸**: 宽度 600px (响应式调整)，高度 48px。
- **搜索框圆角**: 24px (全圆角)。
- **搜索框边框**: 1px solid #E5E7EB。
- **搜索框阴影**: 0 2px 8px rgba(0, 0, 0, 0.1)。

### 2.2 中部标签导航区
- **板块卡片**:
  - 背景: 白色/磨砂玻璃 (backdrop-filter: blur(10px))。
  - 标题: 左对齐，加粗，字号 18px。
  - 内边距: 16px。
  - 圆角: 12px。
  - 间距: 板块之间 24px。
- **服务项网格**:
  - 网格间距: 16px (gap)。
  - 最小宽度: 80px。
  - 图标大小: 32px x 32px。
  - 文字大小: 12px。
  - 文字与图标间距: 8px。

### 2.3 底部 Dock 区
- **位置**: 固定在页面底部，距离底部 16px。
- **背景**: 白色，圆角 16px。
- **内边距**: 12px 16px。
- **网格列数**: 自适应，最小 6 列，最大 10 列。
- **图标卡片**:
  - 尺寸: 56px x 56px。
  - 圆角: 12px。
  - 图标大小: 24px。
  - 文字大小: 10px。
- **"+"号按钮**:
  - 背景: #F3F4F6。
  - 图标: Font Awesome `fa-plus`。
  - 图标大小: 20px。
  - 图标颜色: #9CA3AF。

## 3. 组件样式

### 3.1 搜索标签
- **默认状态**:
  - 文字颜色: #6B7280。
  - 无背景。
  - 字号: 14px。
- **Hover 状态**:
  - 动画: 抖动效果
    ```css
    @keyframes shake {
      0%, 100% { transform: translateX(0); }
      25% { transform: translateX(-2px); }
      75% { transform: translateX(2px); }
    }
    ```
  - 动画时长: 0.3s。
- **Active/Selected 状态**:
  - `transform: translateY(-5px);`
  - `font-weight: 600;`
  - 颜色变亮（对应强调色）。
  - 过渡时间: 0.2s。

### 3.2 模态框
- **遮罩层**:
  - `background: rgba(0, 0, 0, 0.5);`
  - `backdrop-filter: blur(4px);`
- **内容盒**:
  - 宽度: 80vw，最大 800px。
  - 高度: 70vh。
  - 背景: 白色/毛玻璃。
  - 圆角: 16px。
  - 阴影: 0 25px 50px -12px rgba(0, 0, 0, 0.25)。
  - 内边距: 24px。
- **关闭按钮**:
  - 位置: 右上角。
  - 尺寸: 32px x 32px。
  - 圆角: 50%。
  - Hover 背景: rgba(0, 0, 0, 0.1)。

### 3.3 右键菜单
- **背景**: 白色。
- **圆角**: 8px。
- **阴影**: 0 4px 12px rgba(0, 0, 0, 0.15)。
- **菜单项**:
  - 高度: 36px。
  - 内边距: 8px 16px。
  - 字号: 14px。
  - Hover 背景: #F3F4F6。
- **分隔线**: 1px solid #E5E7EB。

### 3.4 确认弹窗
- **背景**: 白色。
- **圆角**: 12px。
- **内边距**: 24px。
- **阴影**: 0 20px 25px -5px rgba(0, 0, 0, 0.1)。
- **按钮**:
  - 取消按钮: 灰色背景 (#F3F4F6)，黑色文字。
  - 确认按钮: 红色背景 (#EF4444)，白色文字。
  - 按钮高度: 36px。
  - 按钮圆角: 8px。
  - 按钮间距: 12px。

### 3.5 AI 对话窗口
- **位置**: 搜索框下方，居中。
- **宽度**: 500px。
- **最大高度**: 400px。
- **背景**: 白色。
- **圆角**: 16px。
- **阴影**: 0 10px 40px rgba(0, 0, 0, 0.2)。
- **消息气泡**:
  - 用户消息: 右对齐，蓝色背景 (#4F46E5)，白色文字。
  - AI 消息: 左对齐，灰色背景 (#F3F4F6)，黑色文字。
  - 圆角: 12px。
  - 内边距: 12px 16px。
  - 最大宽度: 80%。

### 3.6 设置面板
- **位置**: 右侧滑出。
- **宽度**: 320px。
- **背景**: 白色。
- **阴影**: -4px 0 20px rgba(0, 0, 0, 0.1)。
- **内边距**: 24px。
- **标题字号**: 18px，加粗。
- **开关按钮**:
  - 宽度: 44px。
  - 高度: 24px。
  - 圆角: 12px。
  - 开启颜色: #4F46E5。
  - 关闭颜色: #E5E7EB。

### 3.7 图标
- **图标来源优先级**:
  1. `https://www.google.com/s2/favicons?domain=xxx.com&sz=32` (自动获取)
  2. 文字图标: 灰底白字 (#6B7280 背景，白色文字)
  3. 用户选择的 Emoji
- **图标缓存**: 使用 Google Favicon 服务自动缓存。

## 4. 排版
- **主标题**: 24px, Bold, #1F2937
- **搜索框输入**: 16px, Medium, #1F2937
- **卡片标题**: 14px, Medium, #1F2937
- **卡片描述**: 12px, Regular, #6B7280
- **按钮文字**: 14px, Medium
- **字体族**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif

## 5. 响应式断点
- **桌面端**: >= 1024px (默认布局)
- **平板端**: 768px - 1023px
  - 搜索框宽度: 100%，最大 500px
  - Dock 网格: 5-8 列
- **移动端**: < 768px
  - 搜索框宽度: 100%，内边距 16px
  - Dock 网格: 4-6 列
  - 模态框宽度: 95vw
  - 隐藏中部导航区（默认极简模式）

## 6. 动画规范
- **过渡时间**:
  - 快速: 0.15s (hover 状态变化)
  - 正常: 0.3s (模态框、面板展开)
  - 慢速: 0.5s (页面加载动画)
- **缓动函数**:
  - 默认: `ease-in-out`
  - 弹性效果: `cubic-bezier(0.68, -0.55, 0.265, 1.55)`
- **高亮动画**:
  ```css
  @keyframes highlight-pulse {
    0%, 100% { background-color: rgba(239, 68, 68, 0.3); }
    50% { background-color: rgba(239, 68, 68, 0.5); }
  }
  ```
  - 动画时长: 1.5s。
  - 循环: infinite。

## 7. 图标高亮状态
- **高亮背景**: rgba(239, 68, 68, 0.3)。
- **高亮边框**: 2px solid #EF4444。
- **高亮动画**: 脉冲效果，1.5s 循环。
- **清除过渡**: 0.3s 淡出。
